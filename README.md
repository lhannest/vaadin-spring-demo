# vaadin-spring-demo
A demo application showing vaadin and spring

To build use [gradle](https://gradle.org/install/):
```
gradle build
```
Then to run execute:
```
java -jar build/libs/vaadin-spring-demo-0.0.1-SNAPSHOT.jar
```
Open your browser to http://localhost:8080 to see it.
